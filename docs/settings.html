Settings

To change the settings for current session, press "Customize Terminal Emulation" button on the toolbar, or select "Settings" from "Connection" menu. Doing so when there is no open session will change the default settings which will effect all new sessions thereafter.

The " Settings" dialog box includes serveral pages which allow you to view and change different aspect of settings.

  • General Page
  • International Page
  • Terminal Page
  • Display Page
  • Print Page
  • Communication Page
  • Keyboard Page
  • Key Map Page
  • Buttons Page
  • Color Map Page
  • Palette Page
